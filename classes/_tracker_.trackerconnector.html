Class TrackerConnector

Naive implementation of the WebsSocket matchmaking protocol used by WebTorrent services.

Registers at the given server, then returns Peers (simple-peer objects) once they are connected & ready.

    Parameters

      The "ws" or "wss" tracker URL to join.

    • peerID: string

      The ID to identify this client. Reuse this across all trackers.

    • infoHash: string

      The "info hash" to register with the Tracker. This is used as a connection ID.

    • peerConfig: PeerConfig

      An object with additional params to pass into each created simple-peer Peer object.

    • isBlacklisted: Function

      A function, which decides if a given Peer ID may connect pre-handshake.

    • announceInterval: number

      The interval, in milliseconds, that this tracker will re-announce.

    • wantedPeerCount: number

      The total number of peers ideally wanted from this tracker each announce.

    Returns TrackerConnector
