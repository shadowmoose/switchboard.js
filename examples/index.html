Resources

This directory contains a few examples, to demonstrate Switchboard. They can be found listed below.

Examples:

  • Sample Chat App - Demonstrates how peering using a Swarm can be quickly used to set up an effortless, encrypted chat network.
  • Video Streaming Room - This example shares a video file with all connected clients, live as it is played.

Extras

  • Server.js - This is a sample NodeJS webtorrent server that you can run, if you wish, which will create a local tracker server for testing.
